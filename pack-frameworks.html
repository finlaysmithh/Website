Front‑End Frameworks

Goal (≈10 hours)

Set up, build and document a single‑page application using your framework of choice (React, Angular or Vue) with proper state management and routing.

Suggested tools:

  • Node.js & npm
  • Create React App, Angular CLI or Vue CLI
  • API provider (OpenWeather, JSONPlaceholder)
  • ChatGPT

Guided tasks

  1. Set up your project. Install Node and your chosen CLI (Create React App, Angular CLI or Vue CLI). Initialise a new project, run the development server and explore the project structure. Tip: Use npm start to launch a dev server and inspect the generated code.
  2. Build a simple application. Create a small SPA such as a to‑do list, weather dashboard or contact book. Break your UI into reusable components and manage state (useState, services or stores). Fetch data from a public API to populate your app. Tip: Keep components small and use props to pass data; ChatGPT can help debug errors.
  3. Add routing or advanced state management. Implement navigation between at least two views using your framework’s router. Alternatively, integrate a state management library (Redux, NgRx, Pinia) to handle global state. Document how you structured your app and why you chose this approach. Tip: Start simple and gradually introduce libraries when the app becomes complex.
  4. Reflect on your experience. Write a brief reflection discussing what you learned about your chosen framework, the challenges you encountered and how you overcame them. Mention when you might choose this framework over others. Tip: Compare with plain JavaScript or other frameworks if you have experience.

Tip: Use available tutorials, sample projects or ask ChatGPT for guidance as you complete each task.

Need help getting started? Break each task down into small steps, explore free tutorials online and ask ChatGPT to explain unfamiliar concepts. Practise iteratively and document your progress as you go.

What to submit

  • A link to your code repository (GitHub or similar) or a zipped copy of the project
  • A live demo URL if deployed
  • Your reflection document

Send these to Finlay@experience4grads.co.uk. We’ll review your work within 48 hours. If approved, you’ll receive a certificate and LinkedIn endorsement.
