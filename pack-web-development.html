Web Development: HTML, CSS & JavaScript

Goal (≈10 hours)

Plan, design, build and deploy a responsive website using modern web technologies and best practices.

Suggested tools:

  • VS Code or another code editor
  • Browser dev tools & validator services
  • GitHub Pages or Netlify
  • ChatGPT

Guided tasks

  1. Design and build your site. Choose a topic or small business and sketch a site map. Create at least three pages using semantic HTML. Use CSS Flexbox and Grid to craft a responsive layout that adapts to mobile and desktop screens. Tip: Start with mobile‑first design and use media queries to adjust layouts.
  2. Add interactivity with JavaScript. Implement at least two interactive components such as a modal window, accordion, image carousel or form validation. Use event listeners and DOM manipulation to respond to user actions. Tip: ChatGPT can provide code snippets or help debug JavaScript errors.
  3. Optimise and audit. Validate your HTML/CSS with W3C tools, compress images, add descriptive alt attributes and ensure good contrast for accessibility. Use Lighthouse or another audit tool to assess performance and accessibility, then improve any weak areas. Tip: Lazy‑load images and minify CSS/JS where possible.
  4. Deploy and document. Push your site’s code to GitHub and deploy it using GitHub Pages or Netlify. Ensure all links work and your site is accessible via HTTPS. Document your development process and include the live URL in your submission. Tip: A short README.md can describe features, known issues and future improvements.

Tip: Use available tutorials, sample projects or ask ChatGPT for guidance as you complete each task.

Need help getting started? Break each task down into small steps, explore free tutorials online and ask ChatGPT to explain unfamiliar concepts. Practise iteratively and document your progress as you go.

What to submit

  • The live URL to your deployed website
  • Your project files (HTML, CSS, JavaScript)
  • A brief document outlining your design decisions and any challenges

Send these to Finlay@experience4grads.co.uk. We’ll review your work within 48 hours. If approved, you’ll receive a certificate and LinkedIn endorsement.
