Programming Languages

Goal (≈10 hours)

Practise coding fundamentals in multiple languages, compare programming paradigms and strengthen your algorithmic thinking.

Suggested tools:

  • Python, Java and C# compilers or IDEs (VS Code, IntelliJ, Visual Studio)
  • Online judges (LeetCode, HackerRank)
  • Git or version control
  • ChatGPT

Guided tasks

  1. Implement a sorting and statistics program. Choose one language (Python, Java or C#) and write a script that reads a list of numbers from a file, sorts them and outputs basic statistics (mean, median, mode, maximum). Use built‑in libraries where appropriate and write functions to organise your code. Tip: Test your program with different input sizes and ask ChatGPT for help with file I/O or algorithm choices.
  2. Write a data analysis script in Python. Using the same or a different dataset, use Python’s pandas library to compute descriptive statistics and visualise distributions. Save your output to a CSV or chart. Tip: If you’re new to pandas, search for a quickstart guide or ask ChatGPT to explain key functions like groupby and describe.
  3. Translate your program to another language and explore OOP. Rewrite your sorting/statistics script in either Java or C#. While doing so, encapsulate functionality in classes (e.g., a NumberProcessor class). Compare differences in syntax, typing and performance. Tip: Use simple unit tests or a main method to verify your implementation across languages.
  4. Reflect on language differences. Write a short essay (1–2 pages) comparing the languages you used. Discuss ease of use, syntax, performance, library support and suitability for different applications. Provide specific examples from your tasks. Tip: Consider how static vs dynamic typing affected your development process.

Tip: Use available tutorials, sample projects or ask ChatGPT for guidance as you complete each task. For the coding exercises, break problems into small functions, write test cases and search for syntax examples when needed.

Need help getting started? Break each task down into small steps, explore free tutorials online and ask ChatGPT to explain unfamiliar concepts. Practise iteratively and document your progress as you go.

What to submit

  • Your source code files for each language
  • Your Python data analysis notebook or script and any generated charts
  • A comparative reflection document

Compress all files into an archive and send to Finlay@experience4grads.co.uk. We’ll review your work within 48 hours. If approved, you’ll receive a certificate and a LinkedIn endorsement.
