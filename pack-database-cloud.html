Database & Cloud Technologies

Goal (≈10 hours)

Design a relational schema, experiment with a NoSQL database and deploy a small application or database to the cloud, then compare these technologies.

Suggested tools:

  • ER diagram tool (draw.io, Lucidchart)
  • SQL database (SQLite, MySQL, PostgreSQL)
  • NoSQL database (MongoDB, DynamoDB)
  • Cloud platform (AWS/Azure free tier)
  • ChatGPT

Guided tasks

  1. Design a relational database. Choose a domain (e.g., library, inventory, clinic) and draw an ER diagram showing tables and relationships. Implement the schema in your chosen SQL database and populate each table with realistic sample data. Tip: Use foreign keys to enforce integrity and ask ChatGPT for help if you’re unsure about cardinality.
  2. Explore a NoSQL database. Install or use an online NoSQL database like MongoDB Atlas. Create a collection for the same domain and perform CRUD operations (create, read, update, delete). Note how the flexible schema differs from your relational design. Tip: Use aggregation pipelines or simple queries to retrieve documents and compare results.
  3. Deploy to the cloud. Sign up for a free tier on AWS or Azure. Host either your relational database (e.g., using RDS/Azure SQL) or a small application that connects to it. Document the steps with screenshots and note any configuration hurdles. Tip: Consider deploying via Netlify or Heroku for the application layer if easier.
  4. Write a comparison report. Compare relational and NoSQL databases in terms of schema design, scalability, consistency and typical use cases. Summarise the pros and cons of deploying databases and applications in the cloud. Include your own observations from the tasks. Tip: Use a table to present advantages and disadvantages clearly.

Tip: Use available tutorials, sample projects or ask ChatGPT for guidance as you complete each task.

Need help getting started? Break each task down into small steps, explore free tutorials online and ask ChatGPT to explain unfamiliar concepts. Practise iteratively and document your progress as you go.

What to submit

  • Your ER diagram and SQL script
  • Exported collection or screenshots showing NoSQL CRUD operations
  • Documentation of your cloud deployment steps
  • Your comparison report

Email all deliverables to Finlay@experience4grads.co.uk. Your work will be reviewed within 48 hours. If approved, you’ll receive a certificate and LinkedIn endorsement.
